Description:
-----------------
**图片来源**  
http://ng.bdatu.com/  
http://www.nationalgeographic.com/  

项目采用DataBinding处理view与activity间的关系，RxBinding处理view事件  
RxJava+Retrofit封装网络请求，Realm缓存网络数据  
采用RxPermission处理6.0及以上的运行时权限获取  
七牛云对超大图片进行镜像存储并返回缩小后的图片
